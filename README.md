# 🧬 PolynomIA: Analizador de Raíces Multigrado

![Streamlit App](https://img.shields.io/badge/Streamlit-FF4B4B?style=for-the-badge&logo=Streamlit&logoColor=white)
![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![NumPy](https://img.shields.io/badge/NumPy-013243?style=for-the-badge&logo=numpy&logoColor=white)

Una herramienta web interactiva diseñada para ingenieros y estudiantes que permite calcular raíces de polinomios de grado $n$, con soporte completo para **números complejos** y una interfaz optimizada para dispositivos móviles.



## ✨ Características
* **Cálculo Automático:** Obtención de raíces reales e imaginarias instantáneamente.
* **Visualización Científica:** Gráfico de dispersión en el **Plano de Argand** (Plano Complejo).
* **Diseño Responsive:** Interfaz adaptable para smartphones, tablets y desktop.
* **Alta Precisión:** Motor de cálculo basado en la matriz compañera de NumPy.
